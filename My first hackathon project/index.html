Projects

AUTOMATIC ARDUINO CNC DRAWING MACHINE

The computer numerical control (CNC) drawing machine is a computer-controlled machine that receives instructions through a serial port sent from a computer and moves its actuators depending on the received instructions. The goal of this project is to develop a drawing machine from locally available materials which can draw complex sketches on different materials including paper. It will have a significant influence in various educational purpose such as drawing complex PCB layouts, or 3D drawing in fashion and design or engineering. This tool can be a good source of income to artists and fashion designers..
