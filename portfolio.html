Projects

I have a personal python project where I created a School Results Management System(SRMS). This system allows the teachers to input ,retrieve ,edit and delete the results when need arise.This SRMS calculates the total marks for each student, grades them as per the predefined grading system and ranks all the students in ascending order based on their total marks.The same is done for each subject where all the scores in each subject are summed and the totals for each subject are graded and ranked. During retrieval of a student's results an additional bar graph is accompanied to offer a better understanding of a student's performance. The same applies when viewing a particular subject's results.
