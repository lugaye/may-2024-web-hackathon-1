Diabetes Mellitus Application

As a beginner in software development, this is my first ever project made possible through PLP coursework. Diabetes Mellitus Application is an e-commerce platform designed to simplify life for individuals managing diabetes mellitus. This all-in-one digital solution provides a seamless experience for patients to access essential diabetes management equipment, medications, and educational resources.

Role: Sole Developer

Technologies Used:

  • Python
  • Django framework

Key Features (proposed):

  • User-friendly interface for browsing and purchasing diabetes management products
  • Secure patient accounts for managing orders and medical information
  • Educational resources section with up-to-date information on diabetes management
  • Integration of e-commerce functionality for easy online purchases
  • Responsive design for access across various devices

Project Status: In Development (as part of Power Learn Project coursework)

Future Plans:

  • Implement a recommendation system for products based on user profiles
  • Integrate a community forum for patients to share experiences and tips
  • Develop a mobile app version for increased accessibility
